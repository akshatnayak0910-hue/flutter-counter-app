# Flutter Counter App

A simple and elegant Counter App built using Flutter and Dart.

This project demonstrates the basic concepts of Flutter development such as:

* StatefulWidget
* setState()
* Button interactions
* UI design using Container, Row, Column, Stack
* Increment and Decrement functionality

## Features

✔ Increment Counter
✔ Decrement Counter
✔ Clean and modern UI
✔ Responsive design
✔ Simple Flutter project for beginners

## Technologies Used

* Flutter
* Dart
* Material Design

## Output Screen

The app contains:

* AppBar with title
* Counter value display
* Increment button
* Decrement button
* Custom footer text
